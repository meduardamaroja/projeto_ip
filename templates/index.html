Como jogar em colaboração

Dicas rápidas para usar Discord e Steam para jogar com seus amigos.

Usando o Discord

  • Crie um servidor ou entre no servidor do seu grupo.
  • Use um canal de voz para combinar horários e estratégias.
  • Ative a opção de compartilhamento de tela ou Go Live para transmitir seu jogo (pode ser usado como alternativa ao matchmaking).
  • Para jogos cooperativos: convide seu amigo através do botão "Convidar para jogo" (se o jogo integrar com o Discord) ou envie o link/ID do lobby.
  • Dica: use push-to-talk se houver muito ruído para evitar cortes na comunicação.

Usando o Steam

  • Adicione seus amigos na sua lista do Steam e abra a janela de chat com eles.
  • Na página do jogo no Steam, use "Convidar para o jogo" ou o recurso de lobby multiplayer (alguns jogos têm botão no overlay do Steam - Shift+Tab).
  • Use o overlay do Steam (Shift+Tab) para enviar convites rapidamente sem sair do jogo.
  • Se o jogo suportar Remote Play Together, um jogador pode convidar outro para jogar como se estivessem locais — excelente para coop local via internet.
  • Dica: verifique se todos têm versões e DLCs compatíveis para evitar problemas de compatibilidade.
